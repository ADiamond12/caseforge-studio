Build an intralogistics commissioning readiness planner for a synthetic warehouse automation rollout.

The planner should turn issue notes, SQL validation checks, configuration gaps, test results, and handover comments into a structured implementation dossier. It should help a technical lead review readiness before moving a warehouse zone from commissioning into operational handover.

The first release must stay local-first and deterministic. It should produce:
- readiness summary
- blocker list
- owner/action table
- validation checklist
- rollback and handover notes
- export bundle for review

The project must not depend on private workplace data. All examples should use synthetic warehouse, conveyor, station, order-flow, and inventory-check terminology.
